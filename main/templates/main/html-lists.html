HTML Lists

Unordered Lists

Unordered lists use the <ul> tag and display items with bullets by default.

<ul>
  <li>Apple</li>
  <li>Banana</li>
</ul>

Ordered Lists

Ordered lists use the <ol> tag and display items with numbers.

<ol>
  <li>HTML</li>
  <li>CSS</li>
</ol>

Definition Lists

Use <dl>, <dt> (term), and <dd> (definition):

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
</dl>
