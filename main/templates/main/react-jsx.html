JSX & Components in React

What is JSX?

JSX stands for JavaScript XML. It allows us to write HTML in React. JSX makes it easier to write and add HTML in React components.

üí° Note: JSX is not required in React, but it makes the code more readable and concise.

JSX Syntax Example

const element = <h1>Hello, JSX!</h1>;

In JSX, we use angle brackets like HTML. This syntax is then compiled to React.createElement() calls.

Embedding Expressions

You can embed any JavaScript expression in JSX by wrapping it in curly braces {}.

const name = 'Puneeth';
const element = <h1>Hello, {name}!</h1>;

JSX vs HTML Differences

  • Use className instead of class
  • Use htmlFor instead of for
  • Self-close elements like <br /> and <img />

Best Practices

  • Always return one parent element from a component
  • Use meaningful names for components
  • Keep logic out of JSX as much as possible
