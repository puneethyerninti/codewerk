Django Forms

What are Django Forms?

Django Forms help you collect user input and validate it cleanly and securely. They can be rendered with HTML or linked to Django Models directly.

Creating a Form

You can create a custom form by inheriting from forms.Form:

from django import forms

class ContactForm(forms.Form):
    name = forms.CharField(max_length=100)
    email = forms.EmailField()
    message = forms.CharField(widget=forms.Textarea)

Rendering in Templates

In your HTML template, you can render the form using:

<form method="post">
  {% csrf_token %}
  {{ form.as_p }}
  <button type="submit">Send</button>
</form>
Tip: Use {{ form.as_p }} to auto-wrap fields with paragraphs.

Handling Submissions

In your views, handle the submitted form like this:

def contact_view(request):
    if request.method == 'POST':
        form = ContactForm(request.POST)
        if form.is_valid():
            # process data
            ...
    else:
        form = ContactForm()
    return render(request, 'contact.html', {'form': form})

Using ModelForm

You can use forms.ModelForm to create a form directly from a model.

class StudentForm(forms.ModelForm):
    class Meta:
        model = Student
        fields = ['name', 'email']
