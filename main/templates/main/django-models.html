Django Models & Admin

1. What is a Model?

In Django, a model is a Python class that defines the structure of your database table. Each model maps to a single table in the database.

from django.db import models

class Blog(models.Model):
    title = models.CharField(max_length=100)
    content = models.TextField()
üí° Models allow Django to automatically generate database tables and CRUD operations.

2. Registering Models in Admin

To see your models in Django Admin, you must register them in admin.py of your app.

from django.contrib import admin
from .models import Blog

admin.site.register(Blog)
üîí Use the admin panel to manage your models without writing SQL or frontend code.

3. Creating a Superuser

Use this command to create an admin account:

python manage.py createsuperuser
