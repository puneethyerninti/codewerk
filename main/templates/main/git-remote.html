Remote Repositories

What is a Remote Repository?

A remote repository is a version of your project hosted on the internet (like GitHub) that you and your teammates can access and collaborate on.

Adding a Remote

To connect your local repo to GitHub:

git remote add origin https://github.com/yourname/yourrepo.git
üí° "origin" is the default name for your main remote repository.

Pushing to Remote

Use git push to send your commits to GitHub:

git push -u origin main

The -u flag sets "origin/main" as the default for future pushes.

Pulling from Remote

To bring the latest changes from GitHub into your local repo:

git pull origin main

Check Remote URLs

git remote -v

Displays all remotes linked to your repository.
