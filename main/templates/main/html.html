HTML Tutorial

Learn the foundation of every webpage — HTML5 — and how to structure semantic, accessible web content.

Getting Started with HTML

1. What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages.

2. Basic Structure

Every HTML document starts with <!DOCTYPE html>, followed by <html>, <head>, and <body>.

3. Headings & Paragraphs

Use <h1> to <h6> for headings and <p> for paragraphs.

4. Links & Images

<a href=""> creates links. <img src="" alt=""> displays images.

5. Lists

<ul> for unordered, <ol> for ordered, and <li> for list items.
