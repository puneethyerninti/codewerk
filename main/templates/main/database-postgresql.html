Using PostgreSQL with Django

What is PostgreSQL?

PostgreSQL is a powerful, open-source, object-relational database system known for performance, reliability, and feature-rich capabilities.

Installation & Setup

  • Download from postgresql.org
  • Install psycopg2 using: pip install psycopg2-binary

Connect PostgreSQL to Django

Update your settings.py:

DATABASES = {
  'default': {
    'ENGINE': 'django.db.backends.postgresql',
    'NAME': 'codewerk_db',
    'USER': 'postgres',
    'PASSWORD': 'yourpassword',
    'HOST': 'localhost',
    'PORT': '5432',
  }
}

Models & Migrations

Define your models in models.py and apply migrations:

python manage.py makemigrations
python manage.py migrate

Basic Query Examples

# Create
Book.objects.create(title="Codewerk", author="Puneeth")

# Retrieve
Book.objects.filter(author="Puneeth")

# Update
book = Book.objects.get(id=1)
book.title = "Codewerk 2.0"
book.save()

# Delete
book.delete()
