Functions in JavaScript

A function is a reusable block of code designed to perform a particular task. You can define it once and call it multiple times.

üí° Tip: Functions help you keep your code clean, modular, and reusable!

Function Declaration

function greet() {
  console.log("Hello, CODEWERK!");
}

You can call it using:

greet();

Function with Parameters

function greetUser(name) {
  console.log("Hello, " + name + "!");
}

greetUser("Puneeth");

Functions can accept values (called parameters) and use them inside the block.

Return Values

function add(a, b) {
  return a + b;
}

let result = add(5, 3); // result is 8

Functions can return a value using the return statement.
