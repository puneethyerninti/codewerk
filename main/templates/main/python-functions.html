Python Functions

What is a Function?

A function is a block of reusable code that performs a specific task. It helps reduce repetition and makes code easier to manage.

def greet():
    print("Hello from CODEWERK!")
greet()

This will output:

Hello from CODEWERK!

Function Parameters & Arguments

You can pass values into a function using parameters.

def greet(name):
    print("Hello, " + name)

greet("Puneeth")
greet("CODEWERK")

Output:

Hello, Puneeth
Hello, CODEWERK

Return Statement

Functions can return values using the return keyword:

def add(x, y):
    return x + y

result = add(3, 5)
print(result)

Output: 8

Default Arguments

If no value is provided, default arguments can be used:

def greet(name="Guest"):
    print("Hello, " + name)

greet()
greet("Puneeth")

Keyword Arguments

You can pass arguments by name:

def user_profile(name, age):
    print(f"{name} is {age} years old.")

user_profile(age=21, name="Puneeth")
