Loops & Conditions

Conditional Statements

Conditional statements allow your code to make decisions based on conditions using if, else if, and else.

let score = 80;
if (score >= 90) {
  console.log("A grade");
} else if (score >= 75) {
  console.log("B grade");
} else {
  console.log("Needs improvement");
}

Loops

Loops help repeat blocks of code:

// For loop
for (let i = 1; i <= 5; i++) {
  console.log(i);
}

// While loop
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
