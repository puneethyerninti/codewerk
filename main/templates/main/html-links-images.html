HTML Links & Images

1. Creating Links

To create a hyperlink to another page or website, use the <a> (anchor) tag with an href attribute: 

<a href="https://www.codewerk.com">Visit CODEWERK</a>

The text between <a> and </a> is clickable. The href defines the URL.

üí° Tip: Use target="_blank" to open links in a new tab:
<a href="https://example.com" target="_blank">Open in new tab</a>

2. Adding Images

Use the <img> tag with src (source) and alt (alternate text) attributes to display images: 

<img src="logo.png" alt="CODEWERK Logo" />

The alt attribute improves accessibility and is shown if the image fails to load.

You can control image size with width and height attributes or CSS:

<img src="logo.png" width="200" />

3. Link an Image

You can make an image a link:

<a href="https://www.codewerk.com">
  <img src="logo.png" alt="CODEWERK" />
</a>
