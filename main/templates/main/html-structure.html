Basic Structure of an HTML Document

Every HTML page follows a standard structure to tell browsers how to interpret and render its content. Let‚Äôs break down the main parts of an HTML document.

üí° Tip: Always start your document with <!DOCTYPE html>. It ensures modern HTML5 rendering.

HTML Boilerplate:

This is the basic template of a valid HTML5 page:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Page Title</title>
  </head>
  <body>
    <!-- Page content goes here -->
  </body>
</html>

Key Tags Explained

<!DOCTYPE html>: Declares the document type and version (HTML5 in this case).

<html>: Root element that wraps the entire HTML content.

<head>: Contains metadata like title, charset, styles, and scripts.

<title>: Sets the name that appears in the browser tab.

<body>: All the visible content like text, buttons, images, etc., go inside this tag.
