Git Basics

What is Git?

Git is a distributed version control system used to track changes in source code. It helps developers collaborate efficiently by keeping a record of every modification in a project.

Initialize a Git Repository

Use the following command inside your project directory to initialize a Git repository:

git init

Track Files and Stage Changes

After editing files, use git add to stage changes for committing.

git add .
üí° You can also stage specific files like git add index.html

Commit Changes

Once files are staged, commit them with a meaningful message:

git commit -m "Initial commit"

View Commit History

Use this command to see the commit log:

git log
