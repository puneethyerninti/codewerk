CSS Box Model

The CSS Box Model is a fundamental concept that defines how elements are rendered on the web. Each HTML element is considered a box with the following areas:

  • Content: The actual text or image inside the element.
  • Padding: Space around the content, inside the border.
  • Border: The edge around the padding (optional).
  • Margin: The space outside the border that separates elements.

Visual Representation

|--- Margin ---|
|-- Border --|
| Padding |
| Content |

Example CSS

.box {
  margin: 20px;
  padding: 15px;
  border: 2px solid #a020f0;
}

This code sets a 20px margin, 15px padding, and a solid purple border around the element.
