What is CSS?

CSS (Cascading Style Sheets) is a powerful language that describes how HTML elements should be displayed on screen. Without CSS, websites would be plain and unstyled ‚Äî just raw content.

CSS works by associating rules with HTML elements. These rules govern how the content appears ‚Äî its colors, fonts, layout, positioning, and even animations.

üí° Tip: Always separate structure (HTML) from presentation (CSS) for cleaner and maintainable code.

Why Use CSS?

Here are some key benefits of CSS:

  • Maintain consistent design across pages
  • Enable responsive layouts for different devices
  • Use animations and transitions for interactivity
  • Style elements with colors, fonts, shadows, gradients, and more

CSS Syntax Example

h1 {
  color: violet;
  font-size: 32px;
  text-align: center;
}

This rule styles all <h1> headings to have violet text, a size of 32px, and be centered.

Types of CSS

  • Inline CSS ‚Äì added directly to the HTML tag using the style attribute.
  • Internal CSS ‚Äì defined inside a <style> tag in the HTML head.
  • External CSS ‚Äì linked via an external .css file (recommended).
