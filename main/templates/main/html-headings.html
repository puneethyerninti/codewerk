Headings & Paragraphs

HTML Headings

HTML offers six levels of headings, from <h1> (most important) to <h6> (least important).

Use headings to define the structure and hierarchy of your content. Typically, each page should have one <h1>, followed by subheadings like <h2>, <h3>, etc.

Example: 
<h1>Welcome to CODEWERK</h1>
<h2>Frontend Tutorials</h2>
<h3>HTML Basics</h3>

HTML Paragraphs

Paragraphs are created using the <p> tag. Each paragraph block creates a visible space between sections of text.

You should not use multiple <br> tags to create spacing. Instead, wrap text in proper <p> tags.

Example: 
<p>HTML stands for HyperText Markup Language.</p>
<p>It is the standard language of the Web.</p>

Best Practices

  • Use headings to organize your content hierarchically.
  • Only one <h1> per page — for SEO and accessibility.
  • Don’t use headings just for making text bold. Use CSS for styling.
  • Group related content using paragraphs and semantic tags.
