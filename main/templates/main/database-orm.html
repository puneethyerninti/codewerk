Django ORM (Object Relational Mapper)

What is Django ORM?

The Django ORM allows developers to interact with databases using Python objects instead of raw SQL queries.

üí° Example: Instead of SELECT * FROM users, you write User.objects.all().

Basic ORM Queries

# Get all objects
User.objects.all()

# Get a specific user
User.objects.get(id=1)

# Filter records
User.objects.filter(is_active=True)

Creating & Updating Data

# Creating a new user
User.objects.create(username="john", email="john@example.com")

# Updating a user
user = User.objects.get(id=1)
user.email = "new@example.com"
user.save()

Deleting Records

user = User.objects.get(id=2)
user.delete()

This permanently deletes the record from the database.
