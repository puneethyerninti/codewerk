Variables & Data Types in JavaScript

In JavaScript, variables are used to store data values like numbers, text, objects, and more. You can create variables using let, const, or the older var.

üí° Tip: Use const when the value shouldn't change, and let when it should.

Declaring Variables

let name = "Puneeth";
const age = 21;
var city = "Hyderabad";

Here, name, age, and city are variables holding string and number values.

JavaScript Data Types

  • String: Text, written inside quotes. ("Hello")
  • Number: Integer or floating-point. (42, 3.14)
  • Boolean: True or false.
  • Array: A list of values. ([1, 2, 3])
  • Object: Key-value pairs. ({ name: "Puneeth", age: 21 })
  • Null: No value.
  • Undefined: A variable with no assigned value.
