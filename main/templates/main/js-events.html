JavaScript Events & DOM

JavaScript lets you make your website interactive using events like clicks, hovers, and keystrokes. These events can trigger custom functions.

üí° Tip: The DOM (Document Object Model) lets JavaScript access and manipulate HTML content dynamically.

Event Listener Example

document.getElementById("btn").addEventListener("click", function() {
  alert("Button Clicked!");
});

This adds a click event listener to a button with ID btn.

DOM Manipulation

document.getElementById("heading").innerText = "Welcome to CODEWERK";

This changes the text of the element with ID heading.
