Understanding Variables in Python

What are Variables?

Variables in Python are used to store data. You can assign values to a variable using the = operator.

Example:
message = "Hello, World!"
count = 42

Variable Naming Rules

- Variable names must start with a letter or an underscore.
- They can contain letters, numbers, and underscores.
- Python is case-sensitive (data and Data are different).
