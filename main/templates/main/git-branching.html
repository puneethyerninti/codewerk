Git Branching

What is a Git Branch?

A branch in Git is a lightweight movable pointer to a commit. It's a way to work on different features or bug fixes independently from the main codebase.

üí° By default, Git creates a branch named main or master.

Create a Branch

To create a new branch:

git branch feature-x

Switch to a Branch

Use checkout or the newer switch command:

git switch feature-x
git checkout feature-x

List Branches

To view all local branches:

git branch

Merge a Branch

To merge a branch into your current branch (usually main):

git merge feature-x
