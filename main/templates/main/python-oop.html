Object-Oriented Programming in Python

What is OOP?

OOP stands for Object-Oriented Programming. It allows you to structure your code using classes and objects, modeling real-world behavior with attributes and methods.

üí° Example: A car has attributes like color and brand, and methods like drive and stop.

Defining a Class

class Car:
  def __init__(self, brand, color):
    self.brand = brand
    self.color = color

  def drive(self):
    print(f"{self.color} {self.brand} is driving")

my_car = Car("Tesla", "Red")
my_car.drive()

Core Concepts

  • Class: Blueprint for objects
  • Object: Instance of a class
  • __init__(): Constructor method
  • self: Refers to the current instance
  • Method: Function defined in a class

The 4 Pillars of OOP

  • Encapsulation: Hide internal state using classes
  • Abstraction: Only expose necessary details
  • Inheritance: Reuse features of one class in another
  • Polymorphism: Methods behave differently based on object
