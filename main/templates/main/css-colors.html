CSS Colors & Backgrounds

Colors in CSS bring life to your designs. You can define colors in several formats such as named colors, HEX, RGB, RGBA, and HSL.

1. Named Colors

Use color names like red, blue, green.

color: violet;

2. HEX Colors

Start with # followed by 6 hexadecimal characters.

color: #ff6347; /* tomato */

3. RGB & RGBA

Use rgb() or rgba() for transparency control.

background-color: rgb(255, 0, 0);
background-color: rgba(255, 0, 0, 0.5);

4. HSL & HSLA

HSL stands for Hue, Saturation, and Lightness.

color: hsl(120, 100%, 50%);
