React Router

React Router is a powerful library that enables routing in single-page React applications. It allows users to navigate between components without refreshing the page.

Installation

Install React Router using npm:

npm install react-router-dom

Basic Setup

Wrap your app in <BrowserRouter> and define routes using <Route>:


import { BrowserRouter, Routes, Route } from 'react-router-dom';

function App() {
  return (
    <BrowserRouter>
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
      </Routes>
    </BrowserRouter>
  );
}

Linking Between Pages

Use <Link> to navigate:


import { Link } from 'react-router-dom';

<Link to="/about">Go to About</Link>

Dynamic Routing

You can use route parameters for dynamic URLs:


<Route path="/user/:id" element={<UserProfile />} />

Access the id using useParams() hook.
