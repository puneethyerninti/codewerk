React Hooks

What are Hooks?

Hooks are functions that let you “hook into” React features like state and lifecycle methods inside functional components. The most commonly used hooks are useState and useEffect.

useState

useState allows you to add state to a functional component. It returns a stateful value and a function to update it.

import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);
  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>Click me</button>
    </div>
  );
}
Note: You can call useState multiple times to manage multiple states.

useEffect

useEffect lets you run side effects in your components, such as fetching data or setting up event listeners.

import { useEffect } from "react";

useEffect(() => {
  console.log("Component mounted");

  return () => {
    console.log("Component unmounted");
  };
}, []);
Tip: The empty array [] ensures the effect runs only once when the component mounts.
