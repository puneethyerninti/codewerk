Control Flow in Python

If-Else Statements

Python uses if, elif, and else to control conditional execution.

score = 85

if score >= 90:
    print("Excellent!")
elif score >= 60:
    print("Good job!")
else:
    print("Keep practicing!")

For Loop

Use for to iterate over a range or sequence in Python:

for i in range(3):
    print("CODEWERK")

While Loop

The while loop runs as long as the condition is true:

count = 1
while count <= 3:
    print("Looping")
    count += 1
