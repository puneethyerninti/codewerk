SQL Basics

What is SQL?

SQL (Structured Query Language) is the standard language for managing relational databases. It allows you to query, insert, update, and delete data.

SELECT Statement

Used to retrieve data from one or more tables.

SELECT name, age FROM students;

INSERT Statement

Used to insert new rows into a table.

INSERT INTO students (name, age) VALUES ('Akash', 21);

UPDATE Statement

Used to modify existing data.

UPDATE students SET age = 22 WHERE name = 'Akash';

DELETE Statement

Used to delete records from a table.

DELETE FROM students WHERE age > 30;

WHERE Clause

Filters results based on conditions.

SELECT * FROM students WHERE age < 25;

JOIN Statement

Used to combine rows from two or more tables.

SELECT orders.id, customers.name 
FROM orders 
JOIN customers ON orders.customer_id = customers.id;
